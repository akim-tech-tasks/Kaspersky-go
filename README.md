# Task

Метаграмма — это слово, которое получается из исходного при замене одной буквы исходного на другую.  Даны две строки s и t. Вернуть true, если t является метаграммой строки s, иначе вернуть false.

Ограничения:
Строки не содержат в себе знаков пунктуации или пробелов.
Длина строк находится в диапазоне [1,5000]

Пример:

text --> test

кот --> кит

```
func isMetagramm(s, t string) bool {

}
```


Задача со звездочкой:

Определить, является ли слово метаграммой, где заменили N букв исходного слова.

```
func isMetagrammN(s, t string,  n int) bool {

}
```

go run ./cmd/metagram-one
